🎯 درباره خوابنما

خوابنما یک پروژه متن‌باز و دانش‌بنیان است که با هدف democratize کردن پایش سلامت خواب طراحی شده. ما معتقدیم همه افراد بدون توجه به بودجه، deserve دارند کیفیت خواب خود را بشناسند.

این پروژه توسط تیمی از دانشجویان و علاقه‌مندان به هوش مصنوعی و سخت‌افزارهای پزشکی توسعه داده می‌شود.

🖥️ بک‌اند و API

سرور اصلی با FastAPI نوشته شده و مدل Linear Regression روی دیتاست ضربان قلب، دما و صدا آموزش دیده است.

📡 POST /predict - پیش‌بینی زمان کیفیت خواب
📊 GET /health - وضعیت سرویس
Python 3.14 FastAPI Scikit-learn Pandas NumPy
